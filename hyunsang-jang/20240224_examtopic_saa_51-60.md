D : Server Name Indication (SNI)을 사용하도록 구성된 Network Load Balancer로 ALB를 교체하는 것은 유효한 솔루션이 아닙니다. 이것은 HTTP 트래픽을 HTTPS로 리디렉션하는 요구 사항을 해결하지 않을 것입니다.
